About this Project

The whole project took me about 4 monts to finish and in this time i had made about 4 or 5 versions of this page but this is the best i could do.

How this works?
Go to the main page (use the go back button in the side menu)you have to select race and a gender -dragons does not have gender selection-

Features:

  • Five different races to choose.
  • Names are generated randomly.
  • Names and images can be easily related to famous characters from videogames,movies and books of the fantasy genre.
